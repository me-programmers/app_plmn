Pastikan Drive Z: sudah OK

    Cara mengaktifkan Drive Z:
  1. Di menu utama Pentra400, klik icon exit
  2. Pilih change user, pilih OK
  3. Pilih user tech, pilih OK
  4. Kembali ke menu utama, pilih menu diagnostic
  5. Klik logo explorer di kiri bawah (warna kuning).
  6. Klik menu Start, pilih run
  7. Ketik cmd, enter
  8. Ketik net start server, enter
  9. Ketik net start workstation, enter
